## 3. 族群內強勢股篩選

### CANSLIM評分系統 (C-A-N-S-L-I-M)
```python
class CANSLIMStockScorer:
    def calculate_canslim_score(self, stock_data):
        """
        C: 當季每股盈餘成長率 (權重: 20%)
        A: 年度盈餘成長率 (權重: 15%)
        N: 新產品/新管理/新高股價 (權重: 15%)
        S: 供給與需求 (成交量) (權重: 10%)
        L: 領漲股或落後股 (權重: 10%)
        I: 機構投資人持股 (權重: 15%)
        M: 市場方向 (權重: 15%)
        
        總分: 0-100分
        合格線: ≥70分
        """
```

### 台灣市場優化參數
```
1. 營收成長門檻: 季增率 ≥ 20%
2. 股本限制: 50億~5000億台幣
3. 產業集中: 科技/半導體/生技優先
4. 流動性要求: 日均成交量 ≥ 1000張
```

## 5. 綜合排名與推薦

### 綜合評分算法
```python
class CompositeRankingEngine:
    def calculate_composite_score(self, stock_data):
        """
        綜合評分 = 
          產業動能分數 × 30% +
          CANSLIM分數 × 40% +
          技術確認分數 × 30%
        
        調整因子:
          1. 風險調整: 波動率、Beta係數
          2. 流動性調整: 成交量、市值
          3. 分散化調整: 產業集中度
        """
```

### 推薦分級系統
```
S級 (90-100分): 強力買進
  - 強勢產業龍頭股
  - CANSLIM高分 + 技術突破
  - 機構資金持續流入

A級 (80-89分): 買進
  - 強勢產業成長股
  - CANSLIM合格 + 技術確認
  - 有上漲潛力

B級 (70-79分): 觀察
  - 產業動能良好
  - CANSLIM基本合格
  - 需技術面進一步確認

C級 (<70分): 排除
  - 不符合任一條件
```
