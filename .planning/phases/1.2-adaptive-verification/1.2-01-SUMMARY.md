## Key Decisions

### 1. Tiered Floor System (Implemented in `_get_thresholds`)
```python
def _get_thresholds(self, bar: dict) -> tuple[float, float]:
    atr = bar.get("atr")
    if atr and not pd.isna(atr) and atr > 0:
        stop = atr * self._atr_mult_stop
        trail = atr * self._atr_mult_trail
        # Tiered floors: Stop needs 10pt safety, Trail needs 20pt room
        return max(10.0, stop), max(20.0, trail)
    return self._release_stop_fixed, self._trail_dist_fixed
```

**Rationale:**
- Stop floor (10pt): Prevents excessively tight stops that would trigger on noise
- Trail floor (20pt): Gives remaining leg enough room to breathe after release
- Fallback (20pt/30pt): Conservative defaults when ATR is unavailable

### 2. ATR Filter Gate (Implemented in `on_bar`)
```python
if atr is not None and not pd.isna(atr) and atr < self._min_atr:
    self._set_eval(... skip_reason="ATR_TOO_LOW: ...")
    return None  # Skip entry
```

**Rationale:** Low-volatility environments (ATR < 10.0) indicate range-bound markets where spread breakouts are more likely to fail.
